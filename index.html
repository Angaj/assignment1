What is HTML?

HTML stands for Hyper Text Markup Language. It's a markup languag that web developers used to structure and describe the content of a web page (PS: its not a priogramming language)

HTML consist of the elements that describe the different types pof content: Paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, element is made up of 3 parts:

  1. the opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.
